# 资产管理模块集成文档

## 概述

已成功将后端的资产管理功能集成到新的前端应用中，主要包含组织管理功能。

## 文件结构

```
front/
├── app/
│   ├── assets/
│   │   ├── page.tsx                    # 资产管理概览页面
│   │   └── organization/
│   │       ├── page.tsx                # 组织管理页面
│   │       └── [id]/
│   │           └── page.tsx            # 组织详情页面
│   └── layout.tsx                      # 根布局（已包含侧边栏导航）
├── components/
│   └── assets/
│       ├── organization-list.tsx       # 组织列表组件
│       ├── organization-columns.tsx    # 组织表格列定义
│       ├── add-organization-dialog.tsx # 添加组织对话框
│       └── edit-organization-dialog.tsx # 编辑组织对话框
└── ASSETS_INTEGRATION.md              # 本文档
```

## 功能特性

### 1. 资产管理概览页面 (`/assets`)
- 显示资产统计信息
- 功能模块导航
- 快速操作入口

### 2. 组织管理页面 (`/assets/organization`)
- 组织列表展示
- 支持搜索和排序
- 添加、编辑、删除组织
- 批量操作支持
- 响应式设计

### 3. 组织详情页面 (`/assets/organization/[id]`)
- 显示单个组织的详细信息
- 预留扩展空间

## 组件说明

### OrganizationList 组件
- **功能**: 组织数据的展示和管理
- **特性**: 
  - 支持加载、错误、空状态处理
  - 集成数据表格组件
  - 支持搜索和选择
  - 批量操作支持

### OrganizationColumns 组件
- **功能**: 定义组织表格的列结构
- **包含列**: 选择、名称、描述、创建时间、更新时间、操作
- **特性**: 支持排序、自定义渲染

### AddOrganizationDialog 组件
- **功能**: 添加新组织的对话框
- **特性**: 
  - 表单验证
  - 字符计数
  - 加载状态
  - 错误处理

### EditOrganizationDialog 组件
- **功能**: 编辑现有组织的对话框
- **特性**: 
  - 预填充数据
  - 变更检测
  - 表单重置
  - 实时验证

## 使用说明

1. **访问资产管理**: 点击侧边栏的"资产管理"
2. **管理组织**: 进入"组织管理"子菜单
3. **添加组织**: 点击"添加组织"按钮
4. **编辑组织**: 点击表格行的编辑按钮
5. **删除组织**: 点击表格行的删除按钮
6. **查看详情**: 点击组织名称链接
